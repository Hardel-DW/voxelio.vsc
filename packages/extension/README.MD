## Welcome !
Integration of the Misode editor into VS Code, with Spyglass tools for Minecraft datapack editing. This plugin is still in development, so expect bugs and features to be added.
This is a fork of the Misode editor, with the goal of improving the editing experience for Minecraft datapacks.

**All these feature are available**: Support of custom json keys, indentation, CR/LF and Tabs/Space are supported ! 
Enjoy the editor !

![Completions in an NBT tag](https://uy64vwmxhf.ufs.sh/f/llTsukoqRt5QN7pi6Qd6lkEfvG8JidWp0xRAYq7IVcFuU1mH)

## Documentation
Note: Moving to the secondary panel is better for the development experience. Just right click on the editor and select "Move to Secondary Panel".
1. Open a folder containing a `pack.mcmeta` file
2. Click the **Datapack Node Editor** icon in the activity bar
3. Open any JSON file in the `data/` or `/assets` folder to edit it visually

## Commands
- `Mi-Node: Open Node Editor` - Focus the node editor panel
